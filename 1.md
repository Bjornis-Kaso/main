### ⚡ [smart-energy-meter](https://github.com/Bjornis-Kaso/smart-energy-meter)

```
┌─────────────────────────────────────────────────────────────────┐
│  Single-phase AC energy meter — Arduino Uno                     │
│                                                                  │
│  MEASURES  →  Voltage · Current · Active Power · Energy (kWh)  │
│  DISPLAY   →  20×4 I²C LCD · Real-time · No flicker            │
│  ACCURACY  →  < 1.73% error · IEC 62053-21 Class 2             │
│  PROCESS   →  Proteus simulation → Perfboard → Calibration      │
└─────────────────────────────────────────────────────────────────┘
```

**Stack:** `C++` `Arduino` `ACS712-30A` `Proteus` `I²C` `RMS Oversampling` `IEC 62053-21`

**What makes it real:**
- Not a tutorial clone — designed from scratch, component by component
- Voltage divider ratio tuned through physical calibration (÷72.5 final)
- Calibrated against a reference instrument across 10 measurement points
- Firmware went through 3 iterations: V1 prototype → Simulation → V2 production
